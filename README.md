# STWMessagingApp

###  Overview  
A small **iOS messaging app** built with **UIKit** and **Core Data**.  
The goal is to show contact list management, conversations, messages, and sharing of POIs (Points of Interest).

---

###  Main Features
-  **Contacts View**  
  - Display local contacts  
  - Search by first or last name  

-  **Conversations View**  
  - Show saved conversations  
  - Search by contact name  

-  **Messages View**  
  - Send text or image messages  
  - Simulate self-chat (send & receive same message)  
  - Share location or list of POIs  

- **POIs View**  
  - Display nearby points of interest  
  - Show distance and open route in Maps  

-  **Core Data** for saving contacts, messages, and conversations  
- **Localized** in English and French
